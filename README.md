This is website created by Shivam Bourya for RythmicApps


## How to Use
* Open your website in a browser
* Click "Login/SignUp" in the navbar
* Choose SignUp to create an account:
* Enter username, email, password (6+ chars), confirm password
* Click Register
* Modal automatically switches to Login tab
* Enter your credentials and click Login
* Success! Modal closes
